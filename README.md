# react-scaffold
React project scaffolding

## Getting Started
```sh
git clone git@github.com:MentorWebServiceTeam/react-scaffold.git

npm install
```

Open two terminal windows
```sh
# Window 1
npm start

# Window 2
npm run webpack:watch
```

## Tests & Coverage
```sh
npm run test
# or
npm run test:watch
# or
npm run test:cov
```
